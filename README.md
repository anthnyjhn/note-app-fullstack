How to run Server
    Run these terminal commands
        > cd server
        > npm init -y
        > npm i express
        > npm i ejs
        > npm i nodemon
        > npm run dev

How to run Client
    Run these terminal commands
        > cd client
        > npm create vite@latest
        > npm I
        > npm run dev
